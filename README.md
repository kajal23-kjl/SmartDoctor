# Smart Doctor Appointment System

## Overview
Smart Doctor Appointment System is a PHP and MySQL based web application that allows patients to book doctor appointments online easily and efficiently.

The project helps manage:
- Appointment booking
- Patient details
- Doctor selection
- Contact form submissions

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Features
- Online doctor appointment booking
- User-friendly interface
- Database connectivity
- Responsive design
- Contact form management

## Project Structure
- `project/` → Main project files
- `includes/` → Database connection and backend files
- `images/` → Project images/assets
- `register.sql` → Database file
